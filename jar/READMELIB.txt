● Org Apache CommonsHTTPClient: un tool di componenti Java di basso
livello incentrati su protocolli HTTP e associati.

● CommonsLogging : Apache CommonsLogging (precedentemente noto
come Jakarta CommonsLogging, JCL) è un'utilità di registrazione basata su
Java e un modello di programmazione per la registrazione e per altri toolkit.
Fornisce API, implementazioni dei log e implementazioni di wrapper su altri
strumenti.

● iText pdf : iText è una libreria per creare e manipolare file PDF in Java
e .NET. Fornisce supporto per le funzioni PDF più avanzate come le firme
basate su PKI, crittografia a 40 bit e 128 bit, correzione colore, PDF con tag,
moduli PDF (AcroForms), PDF / X, gestione del colore tramite profili ICC e
codici a barre, ed è utilizzato da numerosi prodotti e servizi, tra cui Eclipse
BIRT, Jasper Reports, JBossSeam, Windward Reports e pdftk.

● Commons IO : la libreria IO di Apache Commons contiene classi di utility,
implementazioni di stream, filtri di file, comparatori di file e molto altro.

● CommonsCodec : la libreria Apache CommonsCodec fornisce
implementazioni di codificatori e decodificatori comuni come Base64, Hex,
Phonetic e URL.

● Dom4j 1.5: è una libreria Java open source per lavorare con XML, XPath e
XSLT. È compatibile con gli standard DOM, SAX e JAXP.

● JSON.simple 1.1: JSON.simple è una semplice libreria Java per l'elaborazione
JSON, leggere e scrivere dati JSON e la piena conformità con le specifiche
JSON (RFC4627).

● xChart 1.0.1: xChart è una libreria di grafici Java gratuita che
consente agli sviluppatori di visualizzare e creare i propri grafici.

● MySQL Connector Java : MySQL fornisce driver basati su standard
per JDBC, ODBC e .Net che consentono di creare applicazioni di database.
Inoltre, una libreria C nativa consente agli sviluppatori di incorporare MySQL
direttamente nelle loro applicazioni.

● Jackson Core : il Core Jackson (aka Streaming API), consente di gestire
e manipolare file in formato JSON.

● Unirest-java : Unirest è una libreria client HTTP leggera di Mashape. Insieme a Java, è disponibile anche per Node.js, .Net, Python, Ruby. Fornisce metodi per richieste HTTP

Simple OAuth library for Java

● Scribejava : Scribejava è una libreria di OAuth per java. Consente una autorizzazione API sicura (utilizzata nel caso specifico, per la richiesta a Linkedin)
